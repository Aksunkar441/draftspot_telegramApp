# 🛠 Шаг 3. Как запустить и протестировать всё самому

Мы уже установили зависимости и настроили базу данных для тебя! Тебе осталось выполнить всего несколько простых шагов, чтобы запустить приложение на своем компьютере и открыть его в Telegram.

Давай сделаем это вместе, шаг за шагом.

---

## 🏃‍♂️ Пошаговый запуск

Для запуска нам понадобятся **три окна терминала** (командной строки).

### 🖥 Окно 1: Запуск красивого сайта (Фронтенда)
1. Открой первый терминал.
2. Перейди в папку `frontend`:
   ```bash
   cd frontend
   ```
3. Запусти сервер разработки (Vite):
   ```bash
   npm run dev
   ```
   *Ты увидишь надпись, что сайт запустился на порту 5173 (`http://localhost:5173`). Не закрывай этот терминал!*

---

### 🌐 Окно 2: Создание секретного туннеля (ngrok)
Так как Telegram работает в интернете и требует безопасные ссылки (`https://`), нам нужно показать наш локальный сайт всему миру. Для этого мы используем бесплатную программу **ngrok**.
1. Открой второй терминал.
2. Запусти туннель на порт `5173` (порт нашего фронтенда):
   ```bash
   ngrok http 5173
   ```
3. В окне терминала появится таблица. Найди там строчку, которая начинается с **`Forwarding`** и скопируй ссылку. Она выглядит примерно так:
   `https://a1b2-34-56-78.ngrok-free.app`
   *Держи этот терминал открытым, иначе туннель закроется!*

---

### ✍️ Настройка ссылки в Бэкенде
Прежде чем запускать бэкенд, нам нужно вставить скопированную ссылку ngrok в настройки.
1. Открой файл `backend/.env` (ты можешь сделать это в любом текстовом редакторе).
2. Найди строку `WEBHOOK_BASE_URL` и замени адрес на свою ссылку из ngrok:
   ```env
   WEBHOOK_BASE_URL=https://a1b2-34-56-78.ngrok-free.app
   ```
3. Сохрани файл.

---

### ⚙️ Окно 3: Запуск Умного Судьи (Бэкенда)
Теперь запустим наш сервер на Python, который будет управлять логикой и регистрировать вебхук в Telegram.
1. Открой третий терминал.
2. Перейди в папку `backend`:
   ```bash
   cd backend
   ```
3. Запусти сервер с помощью нашего виртуального окружения (оно лежит в корне проекта):
   ```bash
   ../.venv/bin/uvicorn app.main:app --reload
   ```
   *Бэкенд запустится на порту 8000, свяжется с базой данных и автоматически скажет Telegram отправить сообщения на твой адрес ngrok.*

---

## 📱 Играем в Telegram!

Всё готово! Теперь давай протестируем приложение в Telegram:

1. Открой Telegram на телефоне или компьютере.
2. Найди в поиске нашего бота по имени: **`@draftspot_bot`**
3. Нажми кнопку **Запустить** (или отправь `/start`).
4. **Бот начнет с тобой общаться:**
   * Спросит твое имя. Напиши его.
   * Спросит возраст (напиши цифру, например `20`).
   * Спросит фотографии. Отправь одну или несколько фоток, а затем нажми кнопку **"Готово"** под клавиатурой.
   * Спросит описание профиля. Напиши что-то о себе или нажми **"Пропустить"**.
5. Ура! Регистрация завершена. Бот пришлет тебе кнопку **«Открыть приложение»**.
6. Нажми на нее! Внутри Telegram откроется твой красивый сайт.
   * Ты можешь создать новую игру (кнопка `+` на странице ленты).
   * Выбери спорт, укажи одну из добавленных нами площадок (например, *Центральный стадион Тараз*), введи цену, дату и нажми «Создать».
   * Твой сбор появится в ленте!

---

## ❓ Что делать, если бот не отвечает?
Если ты пишешь боту `/start`, а он молчит:
1. Проверь **Окно 3 (Бэкенд)**. Не выдает ли оно ошибок?
2. Проверь **Окно 2 (ngrok)**. Работает ли туннель? Изменяется ли список запросов на экране, когда ты пишешь боту?
3. Проверь, совпадает ли ссылка в `backend/.env` с той, что запущена в ngrok. Если ты перезапускал ngrok, ссылка меняется! Тебе нужно обновить ее в `.env` и перезапустить бэкенд в Окне 3.
